Team Gamerz

Project Goal

This project was designed to create a typing game where any player can test the overall accuracy and speed of their typing using in a user-friendly interface. Various APIs were used to create categories of random words. A player chooses from four APIs in order to choose which category of words will be used in the game.


There are also varying levels of difficulty so players can increase the speed at which they have to type. The project also includes local storage using firebase in order to populate a high score table. This allows users to sign in with their google accounts and have their scores recorded so they can improve and compete with their classmates.
